Category_Calc(BMI)
  * Recieves a BMI value as a parameter
  * Evaluates the Category the BMI falls under
      * x < 18.5 : Underweight
      * 18.5 <= x < 25 : Normal Weight
      * 25 <= x < 30 : Overweight
      * x >= 30 : Obese
  * Returns a string containing the category
